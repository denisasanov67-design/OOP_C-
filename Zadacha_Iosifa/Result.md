Формула функции для решения задачи Иосифа Флавия выглядит следующим образом: **J(N, k) = (J(N - 1,3) + k - 1) mod N) + 1** 



Формула является рекуррентной, она зависит от количества элементов N и показывает, что положение последнего элемента зависит от решения задачи для предыдущего количества элементов.

&#x20;

В алгоритме последовательно заносятся значения от 0 до N-1. Это требует O(n) операций. Далее удаление элементов из массива требует сдвига всех последющих элементов, поэтому вермя выполнения этой операции тоже заниамает O(n) времени.

График функции неравномерный, имеет пилообразную форму. Это происходит потому, что при увеличении количества элементов структура удаления элементов по кругу периодически изменяется. После определенных значений N происходит резкий скачек значения функции, после чего снова начинается постепенный рост.    
Таким образом, общая вычислительная сложность алгоритма определяется как:
O(n*n) = O(n^2)

Это означает, что при увеличении количества элементов время работы алгоритма растёт квадратично.
данная функция характеризуется алгоритмической сложностью O(N²), поскольку основное время выполнения тратится на многократное удаление элементов из массива и последующий сдвиг оставшихся элементов.
